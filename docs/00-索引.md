# 知识库总索引

欢迎来到你的知识库。这里是所有领域的入口。

## 领域地图

- [💼 工作与项目](工作与项目/index.md)
- [💻 技术与编程](技术与编程/index.md)
- [📚 阅读与学习](阅读与学习/index.md)
- [🌿 生活与兴趣](生活与兴趣/index.md)

## 怎么用

1. **收集**：新笔记先丢进 `Inbox/`（或对应领域的 `Inbox/`）。
2. **整理**：运行自动分类 `python Scripts/classify.py`，脚本会按内容把笔记分到对应领域，并写入 `category` 与 `tags`。
3. **沉淀**：已归位的笔记，在各自领域的分区首页（`index.md`）里用 `[[笔记名]]` 做链接和整理。

## 自动分类规则

脚本按关键词匹配领域（在 `Scripts/classify.py` 的 `DOMAINS` 里可自由增删）：

| 领域 | 触发关键词（示例） |
| --- | --- |
| 工作与项目 | 工作 / 项目 / 会议 / 周报 / 需求 / 客户 / OKR / 复盘 / 立项 |
| 技术与编程 | 代码 / 编程 / 算法 / 前端 / 后端 / 数据库 / Python / 部署 / 调试 |
| 阅读与学习 | 读书 / 阅读 / 书评 / 课程 / 学习 / 文章 / 摘录 / 笔记 |
| 生活与兴趣 | 旅行 / 美食 / 电影 / 音乐 / 爱好 / 灵感 / 生活 / 摄影 |

> 想加新领域？在 `Scripts/classify.py` 的 `DOMAINS` 里加一项，并新建同名文件夹即可。
