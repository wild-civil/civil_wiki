# 分辨率、刷新率与像素密度（PPI）

买屏时最常看到三个数字：分辨率、刷新率、PPI。它们各自衡量什么、又怎么互相牵连？这篇一次讲清。

## 一、分辨率

分辨率 = 屏幕**横向 × 纵向**的像素数。常见档位：

| 名称 | 像素 | 备注 |
|---|---|---|
| HD | 1280×720 | 入门 |
| FHD（1080p） | 1920×1080 | 主流 |
| QHD（2K） | 2560×1440 | 显示器「2K」多指此 |
| 4K（UHD） | 3840×2160 | 电视/高端显示器 |
| 8K | 7680×4320 | 旗舰 |

> 口径提示：显示器圈「2K」多指 2560×1440；电影圈 2K = 2048×1080；手机「2K」常指约 3200×1440。所以别只看「2K/4K」字样，看具体像素数更靠谱。

**长宽比**：16:9（最普遍）、16:10、21:9（带鱼屏）、3:2（笔记本）。

## 二、刷新率

刷新率（Hz）= 屏幕**每秒重绘次数**。60 Hz 是基础，高刷常见 120 / 144 / 240 Hz。

- 越高越流畅，滑动、游戏拖影越少；
- 代价：带宽、功耗、面板成本随之上升（带宽公式见[屏幕接口与带宽](屏幕接口与带宽.md)）。

## 三、PPI（像素密度）

PPI = 每英寸的像素数，直接决定「能不能看出颗粒感」：

$$
\text{PPI} = \frac{\sqrt{\text{横向}^2 + \text{纵向}^2}}{\text{屏幕对角线(英寸)}}
$$

示例：6.1" 手机，分辨率 1080×2340

- √(1080² + 2340²) = √(1.166M + 5.476M) = √6.642M ≈ 2577
- 2577 / 6.1 ≈ **422 PPI**

经验门槛：

- 手机 > 300 PPI，肉眼基本难分颗粒（「视网膜屏」概念门槛约 300）
- 显示器 24" 1080p ≈ 92 PPI；27" 1440p ≈ 109 PPI——**尺寸越大，越要看 PPI 而非分辨率数字**

## 四、观看距离与「刚好够用」的分辨率

人眼分辨极限约 **1 角分**（1/60 度）。在观看距离 $d$（单位：英寸）处，刚好察觉不出像素所需的 PPI 门槛约为：

$$
\text{PPI}_{\min} \approx \frac{3438}{d}
$$

换算成米：$d_{\text{inch}} = d_{\text{m}} \times 39.37$。

| 设备 | 典型距离 | 所需 PPI 门槛（约） | 现实参考 |
|---|---|---|---|
| 手机 | ~25 cm (9.8") | ~350 | 旗舰手机 400+ PPI，绰绰有余 |
| 桌面显示器 | ~60 cm (23.6") | ~146 | 1080p@24" 已难分颗粒 |
| 电视 | ~3 m (118") | ~29 | 4K 在 55" 以上才明显优于 1080p |

**一句话**：别只盯着分辨率数字，结合**尺寸**和**观看距离**看 PPI，才有真实意义。一块 4K 屏如果尺寸很小、贴脸看，和一块大 1080p 电视的「细腻感」要用 PPI 来比，而不是比总像素。
