# 显示面板技术：LCD / OLED / Mini-LED

所有屏幕本质都在做同一件事：**控制每个像素发出多少红、绿、蓝**。区别只在于——光从哪来、怎么控制它。这一篇把主流面板技术的原理和取舍讲清楚。

## 一、两条根本路线

- **需要背光（自身不发光）** → 典型是 **LCD**
- **自发光（每个像素自己发光）** → 典型是 **OLED / Micro-LED**

判断一块屏「黑得纯不纯」，基本就看它属于哪条路线：靠背光漏光的，黑色永远带灰；能直接关掉像素的，才是真黑。

## 二、LCD（液晶显示）

LCD 不是「一块发光的板」，而是好几层叠起来的「光阀门」：

```
[ 背光 LED 灯条 ] → [ 导光板 ] → [ 偏光片 ] → [ 液晶层 ] → [ 彩色滤光片(RGB) ] → [ 偏光片 ]
```

原理：背光发出白光，**液晶本身不发光**，靠电场扭转液晶分子的姿态，控制白光透过多少；再经彩色滤光片把白光滤成 R/G/B 子像素。

按液晶排列方式，常见三种：

| 类型 | 特点 | 典型用途 |
|---|---|---|
| **TN** | 响应快、便宜，但视角窄、色彩差 | 入门显示器、工控 |
| **IPS** | 视角广、色彩准、贵一点 | 手机 / 主流显示器 |
| **VA** | 对比度高、黑色更纯 | 电视、曲面屏 |

- **优点**：技术成熟便宜、没有烧屏、能做很高亮度。
- **缺点**：依赖背光 → 黑色不纯（漏光）；背光层占厚度 → 屏厚；响应不如自发光。

## 三、OLED（有机发光二极管）

每个像素是一个有机材料小灯，**通电自己发光，断电就是纯黑**。

- **优点**：无限对比度（真黑）、轻薄可弯折（柔性屏/折叠屏）、响应快、视角广。
- **缺点**：有机材料会缓慢衰减 → 亮度均匀性下降，长期显示固定图标会**烧屏（burn-in）**留下残影；大尺寸成本高。

> 烧屏不是「用一天就坏」，而是长年累月显示静态元素（状态栏、台标）的累积效应。手机/电视厂商用像素位移、自动隐藏状态栏等手段缓解。

## 四、Mini-LED 与 Micro-LED

- **Mini-LED**：本质还是 LCD，但把背光换成**海量微小 LED** 并做**分区调光（Local Dimming）**——该亮的地方亮、该黑的分区直接关掉，对比度逼近 OLED，又避开了烧屏。是目前电视/高端显示器的热门路线。
- **Micro-LED**：每个像素是独立的**无机** LED，自发光、寿命长、亮度高，被视为「终极形态」；但「巨量转移」工艺极难、成本极高，尚未普及。

## 五、怎么选（对照表）

| 维度 | LCD (IPS/VA) | OLED | Mini-LED | Micro-LED |
|---|---|---|---|---|
| 发光方式 | 背光 | 自发光 | 背光(分区) | 自发光 |
| 对比度 | 中 | 极高 | 高 | 极高 |
| 黑色表现 | 漏光 | 纯黑 | 近纯黑 | 纯黑 |
| 烧屏风险 | 无 | 有 | 无 | 无 |
| 厚度 / 柔性 | 厚 / 否 | 薄 / 可弯 | 厚 / 否 | 薄 / 可 |
| 大尺寸成本 | 低 | 中高 | 中高 | 极高 |

**一句话选型**：日常性价比选 LCD；追求画质与轻薄选 OLED；电视既要高对比又怕烧屏，选 Mini-LED。

> 延伸：屏幕能显示多少种颜色，取决于**色深**（见[屏幕色彩格式与色深](屏幕色彩格式与色深.md)）；这些面板最终靠哪种**接口**把画面传进去，见下一篇[屏幕接口与带宽](屏幕接口与带宽.md)。
