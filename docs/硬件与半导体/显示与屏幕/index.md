# 显示与屏幕

这里收录与屏幕、显示技术、色彩表示相关的内容。从「一个像素到底用多少位来表示颜色」这种最基础的问题开始，逐步扩展到面板、接口与实际工程中的取舍。

## 本章内容

- [屏幕色彩格式与色深（RGB565 / RGB888 等）](屏幕色彩格式与色深.md)：RGB444 / RGB555 / RGB565 / RGB666 / RGB888 到底是什么意思，为什么要分这么多格式，以及嵌入式和桌面里怎么处理它们。
- [显示面板技术：LCD / OLED / Mini-LED](显示面板技术-LCD_OLED_MiniLED.md)：背光 vs 自发光两条路线，TN/IPS/VA、OLED、Mini-LED/Micro-LED 的原理与选型对照。
- [屏幕接口与带宽估算](屏幕接口与带宽.md)：RGB 并行、MIPI DSI、LVDS、eDP 等接口特点，以及带宽怎么算。
- [分辨率、刷新率与像素密度（PPI）](分辨率_刷新率与PPI.md)：分辨率档位、刷新率意义、PPI 计算与观看距离的关系。
- [HDR 与色彩空间（sRGB / P3 / Display-P3）](HDR与色彩空间.md)：色域、亮度单位 nit、HDR 标准，以及把整章串起来的选型思路。
