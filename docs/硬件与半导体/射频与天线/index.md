# 射频与天线

> 射频（RF）与天线是无线产品的命脉：芯片再强，能量送不出去也是白搭。这一章把「天线怎么选、怎么画、怎么匹配、怎么避开那些坑」讲清楚，并延伸到各个领域里天线的实际用法。

2.4 GHz ISM 频段（2.4–2.4835 GHz）是 WiFi / BLE / Zigbee / Thread 共用的「免费」频段，也是绝大多数入门射频设计的起点，所以本章从它切入，再横向铺开到 GPS、蜂窝、车载等更宽的领域。

## 本章内容

- [2.4G 天线设计方案](2.4G天线设计方案.md)：选拓扑 → 50Ω 阻抗匹配 → PCB 布局，附方案分类图与尺寸/匹配图，以及最常见的坑与设计流程。
- [天线类型与各领域应用](天线类型与各领域应用.md)：单极子 / 偶极子 / 倒 F / 陶瓷 / 螺旋 / 贴片各自的原理、优缺点，以及在 WiFi、BLE、GPS、蜂窝、车载等领域的实际取舍。
- [射频传输线：微带与共面波导（CPW / GCPW）](射频传输线-微带与共面波导.md)：先铺开有哪些传输线，再深挖共面波导，并专门对比两层板 vs 多层板两种处境下的阻抗、过孔与净空做法。

## 规划中的主题

- 史密斯圆图（Smith Chart）入门：怎么读、怎么用它调匹配
- 巴伦（Balun）与差分 RF 接口：nRF / ESP32 的差分转单端
- 射频 PCB Layout 实战：净空、50Ω 微带线、地平面、过孔
- 天线实测：VNA 看 S11、暗室测增益、OTA 吞吐与距离
