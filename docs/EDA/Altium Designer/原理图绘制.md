# Altium Designer 原理图绘制

原理图（Schematic）是电路的「逻辑蓝图」：它不关心板子长什么样，只关心「谁和谁电连接」。画对原理图，后面的 PCB 才不会连错线。

> 前置：先读 [基础知识](基础知识.md)，弄清工程、网络、元件、封装等概念。

## 步骤总览

1. 在工程里新建/打开 `.SchDoc`；
2. 从库放元件；
3. 用导线、网络标签把元件连起来；
4. 标注（Annotate）自动编号；
5. 编译工程，跑 ERC 查错；
6. 同步到 PCB（Update PCB）。

## 1. 放元件

- 打开 **Components** 面板，搜索并放置（Place）元件到图纸。
- 每个元件有 **Designator**（标号，如 R1、U2、C3）和 **Comment**（值/型号）。
- 放好后按 `R` 旋转、`X/Y` 翻转方向。

## 2. 连线：导线 vs 网络标签

两种方式表达「这两个点连在一起」：

- **导线（Wire）**：直接拉线连接相邻引脚，最直观。
- **网络标签（Net Label）**：在两条网络上各放同名标签，就表示它们属于同一网络——适合**跨处/跨页**连接，不必拉一条长飞线。

> 同名 Net Label = 同一 Net。这是多图设计的基础，务必保证拼写一致（大小写敏感）。

电源与地用 **Power Port**（VCC / GND 等符号），本质也是网络。

## 3. 总线（Bus）

并行信号（如数据总线 `D0..D7`、地址线）可用 **Bus** 加 **Bus Entry** 表示成一股，内部用 Net Label 命名 `D0`…`D7`。它只是「画起来整洁」，电气连接仍靠 Net Label 名字。

## 4. 跨图纸连接

板子大了原理图会分多张，跨张连接用：

- **Port（端口）**：图纸间的网络出入口；
- **Off-Sheet Connector（离图连接器）**：同一工程内跨子图连接。

> 层次化设计（Hierarchical）用 Sheet Symbol + Port 表达模块关系，进阶再学。

## 5. 标注（Annotate）

放完元件后统一编号：

- `Tools → Annotation → Annotate Schematics` 自动按规则编号（R1,R2… U1,U2…）。
- 标注前元件标号是 `R?` 这样的占位，编译会报错，标注后才合法。

## 6. 编译与 ERC

- `Project → Compile` 编译工程。
- **ERC（电气规则检查）** 会抓：未连接引脚（Floating）、重复标号、单端网络（Net 只接了一端）、电源/地冲突等。
- 在 **Messages** 面板看错误，逐个修。常见坑：**网络标签拼写错导致两段没连上**、**忘了放 GND**。

## 7. 同步到 PCB

原理图确认无误后：

- `Design → Update PCB Document`（或在 PCB 里 `Import Changes`），生成 **ECO（工程变更单）**；
- 执行 ECO，原理图的网络、元件就进到 PCB 里（此时是一堆带飞线的元件）。

同时可输出：

- **BOM（物料清单）**：`Reports → Bill of Materials`；
- 网络表（通常不用手导，Update PCB 已包含）。

## 好习惯

- 电源、地、关键信号命名清晰，方便后面布线找网络；
- 模块分区摆放，用放置框（Blanket/Note）标注；
- 能用 Net Label 跨接就别拉超长导线；
- 标注前先编译，把 ERC 清零再进 PCB。

## 下一步

原理图画对后，进入 [PCB Layout 基础知识](PCB Layout 基础知识.md)，把元件布到板子上、把线走通。
