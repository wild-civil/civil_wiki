# Altium Designer

Altium Designer 是业界使用最广的专业 PCB 设计工具之一，擅长中高复杂度板卡：BGA、DDR、高速差分、射频、刚柔结合等。它是付费订阅制，但岗位认可度高、资料最全，是走硬件职业路线绕不开的一站。

> 本分支先打底：先读 **基础知识** 建立术语与界面认知，再读 **原理图绘制** 把电路画对，最后读 **PCB Layout 基础知识** 把板子布出来。三篇是一个连贯的入门路径。

## 工程文件结构

| 扩展名 | 是什么 |
|---|---|
| `.PrjPcb` | 工程文件，把原理图、PCB、库、输出串在一起 |
| `.SchDoc` | 原理图文档（一张电路图） |
| `.PcbDoc` | PCB 文档（一块板子） |
| `.SchLib` | 原理图库（画元件符号） |
| `.PcbLib` | PCB 库（画封装/焊盘） |
| `.IntLib` | 集成库（符号+封装打包） |
| `.OutJob` | 输出任务（一键生成 Gerber/BOM/坐标） |

## 设计流程（总览）

1. **建工程**：新建 `.PrjPcb`，挂上原理图与 PCB 文档。
2. **画原理图**：放元件、连线（网络），标注，编译查 ERC。
3. **导入 PCB**：把原理图网络表同步到 PCB（工程变更单 ECO）。
4. **布局（Placement）**：摆放器件，模块化、考虑散热与装配。
5. **布线（Routing）**：走线、过孔、差分对、铺铜。
6. **规则检查（DRC）**：按设计规则查间距/线宽/孔径等。
7. **输出生产文件**：Gerber、NC Drill、BOM、坐标文件（OutJob 一键出）。
