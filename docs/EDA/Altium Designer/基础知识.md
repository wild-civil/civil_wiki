# Altium Designer 基础知识

在动手画板前，先把界面、单位、工程结构和一批核心术语搞清，后面画原理图和布 PCB 会顺很多。

## 界面与工作区

启动后核心是这几块：

- **Projects 面板**：管理工程里的所有文档（原理图、PCB、库）。
- **Components 面板**：从库里搜元件、放元件。
- **Properties 面板**：看/改当前选中对象的属性（最常用）。
- **PCB / Schematic 编辑器**：主画布，原理图和 PCB 各一个。
- 顶部工具栏 + 快捷键：常用操作都有快捷键，熟练后几乎不离开键盘。

> 建议第一件事：新建一个 `.PrjPcb` 工程，再往里加 `.SchDoc` 和 `.PcbDoc`。**永远在工程里工作**，跨文档功能（ERC、网络同步）才有效；单独打开一个 `.SchDoc` 很多功能用不了。

## 单位：mil 还是 mm

PCB 圈普遍用 **mil**（千分之一英寸，1 mil = 0.0254 mm）。线宽、间距、焊盘常以 mil 说话（如 10 mil 线宽、6 mil 间距）。

- 切换单位：`View → Toggle Units`，或快捷键 `Q`。
- 入门建议先习惯 mil，看资料和嘉立创工艺要求时对得上。

## 工程 vs 文档

| 概念 | 说明 |
|---|---|
| 工程（Project） | `.PrjPcb`，把多张图、PCB、库组织在一起，提供 ERC/同步能力 |
| 文档（Document） | 单张 `.SchDoc` / `.PcbDoc`，工程的一部分 |

## 层（Layers）速览

PCB 是「层的堆叠」，常用层：

- **信号层**：Top / Bottom，放走线；多层板还有 Mid（内层）。
- **电源/地平面（Plane）**：整层铜皮当电源或地，低阻抗。
- **丝印层（Silkscreen）**：板子表面的白字/标号。
- **阻焊层（Solder Mask）**：绿油开窗，露出要焊接的焊盘。
- **钢网层（Paste Mask）**：SMT 钢网开口，用于贴片。
- **机械层（Mechanical）**：板框、尺寸标注。
- **Keep-Out**：禁止布线/放置区。

> 双层板默认只有 Top + Bottom 两个信号层，足够绝大多数入门和中低复杂项目。

## 核心术语

- **网络（Net）**：被「电连接」在一起的一组引脚，原理图里用导线/网络标签定义，PCB 里体现为需要连起来的飞线。
- **元件（Component）**：一个真实器件 = 原理图符号 + PCB 封装。
- **封装（Footprint）**：器件在板子上的物理形状——焊盘位置、尺寸、丝印。
- **焊盘（Pad）**：元件引脚在板上的可焊金属，分通孔（Through-hole）和表贴（SMD）。
- **过孔（Via）**：连接不同信号层的金属孔。
- **丝印（Silkscreen）**：板面标识，方便焊接和维修。

## 库的概念

- **原理图库 `.SchLib`**：画元件的「逻辑符号」（引脚、标号）。
- **PCB 库 `.PcbLib`**：画元件的「封装」（焊盘、外形）。
- **集成库 `.IntLib`**：把符号和封装打包，分发方便。

> 实际项目里，元件 = 符号指向某个封装。Altium 自带常用库，也能从制造商（如 SnapEDA、立创）下载现成库。封装画错是最常见的「板子回来焊不上」的原因，后面会专门讲。

## 栅格与吸附

画布有栅格（Grid），元件和线默认吸附到栅格点。**电气栅格（Electrical Grid）** 会在你靠近引脚时自动吸附，方便连线。快捷键 `G` 可切换栅格步长。
