# EDA（电子设计自动化）

EDA（Electronic Design Automation，电子设计自动化）是指从「电路想法」到「能拿去工厂生产的文件」这一整套软件工具链：画原理图 → 画 PCB → 输出 Gerber / BOM / 坐标文件。它是硬件工程师把图纸变成实物的核心生产力工具。

> 这一章不绑定某一款软件，而是按「工具」来组织：先认清市面上有哪些 EDA、各自适合什么场景，再深入具体工具。当前先开 **嘉立创 EDA** 与 **Altium Designer** 两条分支，后续按需补 KiCad、Cadence 等。

## 工具全景

| 工具 | 定位 | 授权 | 适合谁 |
|---|---|---|---|
| 嘉立创 EDA（LCEDA） | 免费、云端+客户端，与嘉立创打样/SMT 深度打通 | 免费 | 入门、国产打样、想「画完直接下单」 |
| Altium Designer | 业界标准，中高复杂度板卡 | 订阅付费 | 专业硬件、高速/刚柔结合/复杂板 |
| KiCad | 开源免费、功能强、社区活跃 | 免费 | 进阶爱好者、预算敏感的小团队 |
| Cadence Allegro / OrCAD | 高速/射频/IC 级 | 企业授权 | 大厂、高速信号、封装/IC 协同 |
| Eagle → Fusion 360 | 老牌、现并入 Autodesk | 订阅 | 爱好者、机械+电子一体 |

## 怎么选

- 想最快把板子做出来、且主要在嘉立创打样：**嘉立创 EDA**，画完一键下单。
- 想认真做硬件、预算有限：**KiCad**，能力接近商业软件。
- 要做复杂板（BGA、DDR、射频、刚柔结合）或走职业路线：**Altium Designer**，资料最多、岗位最认。
- 高速/射频/企业级协同：**Cadence**。

## 本章分支

- [嘉立创 EDA](嘉立创EDA/index.md)：免费云端工具，从入门到「画完直接下单」。
- [Altium Designer](Altium Designer/index.md)：专业工具，当前包含 **基础知识 / 原理图绘制 / PCB Layout 基础知识** 三块。

## 规划中的分支

- KiCad：开源旗舰，与 Altium 的对照与迁移
- Cadence Allegro：高速板与 IC 协同
- 通用主题：封装库管理、DFM（可制造性）、阻抗计算、叠层设计

## 阅读建议

如果是第一次系统性学 EDA，建议：**先读 Altium Designer 的「基础知识」打术语底 → 「原理图绘制」把电路画对 → 「PCB Layout 基础知识」把板子布出来**。嘉立创 EDA 可以并行上手，因为它更「开箱即用」，适合边学边打样。
