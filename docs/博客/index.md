# 博客 / Blog

这里记录日常的零碎观点、随笔与阶段性小结——把博客当 gist 用，不必像知识库正文那样字斟句酌。

## 写作约定

- 每篇放在 `docs/博客/` 下，文件名以日期开头：`YYYY-MM-DD-标题.md`。
- 文首加 frontmatter：

  ```yaml
  ---
  title: 文章标题
  date: 2026-07-17
  tags:
    - 随笔
  ---
  ```

- 左侧导航会自动按本分区列出全部随笔，最新的在最上方。

> 本页与示例随笔均为演示占位，可按需删除，真实随笔直接新增 `YYYY-MM-DD-*.md` 即可。
