# 发动机二冲程与四冲程

> 内燃机（Internal Combustion Engine）把燃料的化学能通过燃烧直接转化为机械能。按**完成一个工作循环曲轴所需的转数**，最常见的分类就是**四冲程**与**二冲程**。两者本质区别在于：四冲程用「气门」控制进排气、曲轴转两圈做一次功；二冲程靠「活塞开关扫气口 / 排气口」、曲轴转一圈就做一次功。

## 一、基本概念

- **冲程（Stroke）**：活塞从上止点（TDC）到下止点（BDC），或反之，的一次单向运动。
- **工作循环**：进气 → 压缩 → 做功 → 排气 四个过程合称一个循环。
- **上 / 下止点**：活塞离曲轴最远 / 最近的位置。
- **压缩比 $\varepsilon$**：$\varepsilon = \dfrac{V_c + V_s}{V_c}$，即气缸总容积与燃烧室容积之比，决定热效率上限。

| 项目 | 四冲程 | 二冲程 |
| :--- | :---: | :---: |
| 一个工作循环曲轴转数 | 2 圈（720°） | 1 圈（360°） |
| 做功次数（每循环） | 1 次 | 1 次 |
| 同转速下做功频率 | 基准 | 约 2 倍 |
| 配气方式 | 气门（凸轮轴驱动） | 扫气口 + 排气口（活塞控制） |
| 润滑方式 | 独立油底壳机油循环 | 机油混入燃油（或单独机油泵烧掉） |

## 二、四冲程工作循环

曲轴每转两圈完成「进气—压缩—做功—排气」四个冲程，只有**做功冲程**输出动力。

```mermaid
flowchart LR
    A["① 进气<br>活塞↓ · 进气门开<br>吸入空气/混合气"] --> B["② 压缩<br>活塞↑ · 两门关闭<br>压缩混合气"]
    B --> C["③ 做功<br>点火/压燃 · 燃气膨胀<br>推活塞↓（唯一动力冲程）"]
    C --> D["④ 排气<br>活塞↑ · 排气门开<br>排出废气"]
    D --> A
```

1. **进气冲程**：活塞下行，进气门开、排气门关，气缸内压力低于大气压，吸入空气（柴油机）或空气—燃油混合气（汽油机）。
2. **压缩冲程**：活塞上行，两气门全闭，混合气被压缩、温度压力升高。
3. **做功冲程**：接近上止点点火（汽油机火花塞 / 柴油机压燃），高温燃气膨胀推活塞下行，经连杆带动曲轴输出扭矩。
4. **排气冲程**：活塞上行，排气门开，废气被推出气缸。

> 四冲程的进、排气完全由**配气机构**（凸轮轴 + 气门 + 正时皮带 / 链条）精确控制，正时准确，但机构复杂、零件多。

## 三、二冲程工作循环

曲轴每转**一圈**就完成一次做功，活塞上行与下行各承担半个循环，没有独立的「进气 / 排气」冲程——它们被压缩到活塞接近下止点时，由端口开闭顺便完成。

```mermaid
flowchart LR
    E["上行 · 压缩<br>气缸内压缩<br>曲轴箱同时产生负压吸气"] --> F["上止点 · 点火做功<br>燃气膨胀推活塞↓"]
    F --> G["下行 · 扫气<br>先露排气口排废气<br>再露扫气口 · 曲轴箱预压混合气扫入"]
    G --> E
```

1. **压缩（活塞上行）**：气缸顶部混合气被压缩；同时活塞下方曲轴箱容积增大、形成负压，经进气口（或簧片阀）吸入新鲜混合气。
2. **点火做功（上止点附近）**：火花塞点火，燃气膨胀推活塞下行。
3. **排气 + 扫气（活塞下行）**：活塞先露出**排气口**，废气靠自身压力冲出；继续下行露出**扫气口**，曲轴箱内已被预压的新鲜混合气涌入气缸，一边填充一边把残余废气「扫」出排气口。

> 二冲程没有气门，靠**活塞裙部**遮挡 / 打开缸壁上的端口来配气，结构简单、重量轻。代价是扫气时部分未燃混合气会随废气直接跑掉（「扫气短路损失」），所以油耗与 HC 排放偏高。

## 四、核心差异对比

| 维度 | 四冲程 | 二冲程 |
| :--- | :--- | :--- |
| 做功频率 | 低（每 2 圈 1 次） | 高（每 1 圈 1 次） |
| 结构复杂度 | 高（气门、凸轮、正时） | 低（无气门、无配气机构） |
| 重量 / 体积 | 大 | 小、轻 |
| 燃油经济性 | 好 | 差（扫气短路损失） |
| 排放 | 低 | 高（烧机油 + 短路损失） |
| 低速扭矩 | 平稳、线性 | 易「两头弱中间强」，扭矩波动大 |
| 润滑 | 独立机油循环，不烧机油 | 机油参与燃烧，需定期补机油 |
| 维护 | 正时 / 气门需保养 | 极简，但火花塞、活塞环损耗快 |
| 散热压力 | 较低（做功间隔长） | 高（持续做功，易过热） |

!!! tip "为什么二冲程「力气大却更费油」"
    同排量、同转速下二冲程做功次数是四冲程的两倍，因此**升功率**高、低速爆发力强；但扫气过程中一部分新鲜混合气没参与燃烧就排走了，能量利用率低，所以实际油耗反而更高。

## 五、典型应用

| 场景 | 偏好 | 代表产品 |
| :--- | :--- | :--- |
| 汽车 / 乘用车 | 四冲程 | 几乎全部汽油 / 柴油车 |
| 现代摩托车 | 四冲程 | 街车、踏板、ADV |
| 园林 / 手持动力工具 | 二冲程 | 链锯、割草机、吹风机 |
| 舷外机 / 小艇 | 二冲程（多） | 船用挂机 |
| 航模 / 小飞机 | 二冲程 | 模型内燃机 |
| 大型船用 / 发电 | 四冲程 / 低速二冲程柴油 | 船用低速机（二冲程压燃）、机组 |

> 现代法规要求严格后，二冲程在road vehicle（道路车辆）上基本被四冲程取代；但在**功率重量比**敏感的便携工具、舷外机、航模领域仍是主流。

## 六、一句话总结

- **四冲程**：结构复杂但省油、清洁、平顺，适合需要长寿命和合规排放的车辆。
- **二冲程**：结构简单、轻、爆发力强，适合对重量敏感、维护极简的手持 / 小型动力设备。
