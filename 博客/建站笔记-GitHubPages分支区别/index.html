    GitHub Pages е»әз«ҷ ж•ҷзЁӢ

    е»әз«ҷз¬”и®°пјҡGitHub Pages зҡ„ main дёҺ gh-pages еҲҶж”ҜеҲ°еә•д»Җд№Ҳе…ізі»¶

    жҠҠйқҷжҖҒз«ҷйғЁзҪІеҲ° GitHub Pages ж—¶пјҢд»“еә“йҮҢеҮ д№ҺжҖ»дјҡеҮәзҺ°дёӨдёӘеҲҶж”Ҝпјҡmain е’Ң gh-pagesгҖӮ еҫҲеӨҡдәәпјҲеҢ…жӢ¬жҲ‘пјүдёҖејҖе§ӢйғҪжҗһж··иҝҮвҖ”вҖ”жҠҠ Pages зҡ„ Source иҜҜи®ҫжҲҗ mainпјҢз»“жһңз«ҷзӮ№ 404гҖҒ иҮӘе®ҡд№үеҹҹеҗҚзҡ„ HTTPS д№ҹиө·дёҚжқҘгҖӮиҝҷзҜҮжҠҠиҝҷдёӨдёӘеҲҶж”Ҝзҡ„еҲҶе·ҘгҖҒPages зҡ„е·ҘдҪңеҺҹзҗҶгҖҒд»ҘеҸҠжңҖеёёи§Ғзҡ„еқ‘и®Іжё…жҘҡгҖӮ

    дёҖгҖҒе…ҲиҜҙз»“и®ә¶

    • main = жәҗеҲҶж”Ҝпјҡж”ҫгҖҢеҺҹж–ҷгҖҚвҖ”вҖ”MarkdownгҖҒmkdocs.ymlгҖҒconfig.tomlгҖҒHugo жәҗж–Үд»¶гҖҒеӣҫзүҮжәҗзӯүгҖӮ е®ғзҡ„ж №зӣ®еҪ•жІЎжңүеҸҜзӣҙжҺҘжү“ејҖзҡ„ index.htmlпјҢдёҚиғҪеҪ“жҲҗе“ҒзҪ‘з«ҷз”ЁгҖӮ
    • gh-pages = жһ„е»әдә§зү©еҲҶж”Ҝпјҡж”ҫгҖҢжҲҗе“ҒгҖҚвҖ”вҖ”mkdocs build / hugo з”ҹжҲҗзҡ„ HTML / CSS / JSгҖӮ иҝҷжүҚжҳҜзңҹжӯЈиғҪиў«жөҸи§ҲеҷЁи®ҝй—®зҡ„зҪ‘з«ҷгҖӮ
    • GitHub Pages зҡ„ Source еҝ…йЎ»жҢҮеҗ‘ gh-pagesпјҲеҚіеҗ«жҲҗе“Ғзҡ„еҲҶж”ҜпјүпјҢдёҚжҳҜ mainгҖӮ йҖүй”ҷ вҶ’ 404пјҢеӣ дёә main йҮҢж №жң¬жІЎжңүиғҪжңҚеҠЎзҡ„йҰ–йЎөгҖӮ

    дёҖеҸҘиҜқпјҡmain жҳҜдәәеҶҷзҡ„пјҢgh-pages жҳҜжңәеҷЁз”ҹжҲҗзҡ„пјӣPages еҸӘи®Ө gh-pagesгҖӮ

    дәҢгҖҒGitHub Pages жҖҺд№Ҳе·ҘдҪңзҡ„¶

    GitHub Pages жң¬иҙЁдёҠжҳҜдёҖдёӘгҖҢйқҷжҖҒж–Үд»¶жңҚеҠЎеҷЁгҖҚпјҡдҪ е‘ҠиҜүе®ғ

    гҖҢеҺ» жҹҗдёӘеҲҶж”Ҝ зҡ„ жҹҗдёӘзӣ®еҪ• дёӢпјҢжҠҠж–Үд»¶еҪ“зҪ‘з«ҷж №зӣ®еҪ•serve еҮәеҺ»гҖӮгҖҚ

    • еҪ“дҪ еңЁ Settings вҶ’ Pages вҶ’ Source йҖү gh-pages / rootпјҲжҲ– docs/пјүж—¶пјҢ GitHub е°ұдјҡжҠҠйӮЈдёӘеҲҶж”ҜйҮҢзҡ„ж–Үд»¶зӣҙжҺҘжҳ е°„жҲҗ https://<user>.github.io/<repo>/пјҢ еҶҚеҠ дёҠдҪ й…Қзҡ„иҮӘе®ҡд№үеҹҹеҗҚпјҲеҰӮ nav.hanvon.topпјүгҖӮ
    • е®ғдёҚдјҡеё®дҪ жһ„е»әгҖӮеҰӮжһңеҲҶж”ҜйҮҢеҸӘжңү .md жәҗж–Үд»¶гҖҒжІЎжңү .htmlпјҢи®ҝе®ўе°ұеҸӘдјҡжӢҝеҲ°дёҖе Ҷ жәҗз ҒжҲ– 404пјҢиҖҢдёҚжҳҜзҪ‘з«ҷгҖӮ

    жүҖд»ҘгҖҢжәҗз ҒеҲҶж”ҜгҖҚж°ёиҝңдёҚиғҪзӣҙжҺҘеҪ“ SourceгҖӮдёӯй—ҙеҝ…йЎ»жңүдёҖйҒ“гҖҢжһ„е»әгҖҚжҠҠжәҗеҸҳжҲҗ HTMLпјҢ иҖҢ gh-pages е°ұжҳҜиҝҷйҒ“жһ„е»әзҡ„дә§еҮәиҗҪзӮ№гҖӮ

    еӣӣгҖҒиҮӘе®ҡд№үеҹҹеҗҚпјҲCNAMEпјүж”ҫе“Ә¶

    иҮӘе®ҡд№үеҹҹеҗҚиҰҒеҮәзҺ°еңЁжҲҗе“Ғж №зӣ®еҪ•зҡ„ CNAME ж–Үд»¶йҮҢпјҢPages жүҚиғҪзӯҫеҸ‘иҜҒд№Ұе№¶з”ҹж•ҲгҖӮ дҪҶгҖҢжәҗгҖҚйҮҢзҡ„дҪҚзҪ®еӣ еј•ж“ҺиҖҢејӮпјҡ

    еј•ж“Һ CNAME еңЁжәҗйҮҢж”ҫе“Ә жҖҺд№ҲеҲ°жҲҗе“Ғж №
    MkDocs docs/CNAME mkdocs build жҠҠ docs/ ж•ҙдҪ“зј–иҜ‘пјҢCNAME з•ҷеңЁж №
    Hugo static/CNAME Hugo жҠҠ static/ еҺҹж ·еӨҚеҲ¶еҲ°з«ҷзӮ№ж №

    ж”ҫй”ҷдҪҚзҪ®зҡ„е…ёеһӢз—ҮзҠ¶пјҡgh-pages ж №жІЎжңү CNAME вҶ’ иҮӘе®ҡд№үеҹҹеҗҚ HTTPS еӨұж•ҲпјҢеӣһйҖҖеҲ° github.ioгҖӮ

    дә”гҖҒжңҖеёёи§Ғзҡ„еқ‘пјҡSource йҖүжҲҗдәҶ main¶

    з—ҮзҠ¶пјҡhttps://<иҮӘе®ҡд№үеҹҹеҗҚ> и®ҝй—® 404пјҢжҲ– HTTPS жҸЎжүӢеӨұиҙҘпјҲTLS иө·дёҚжқҘпјүгҖӮ

    ж №еӣ пјҡеңЁ Settings вҶ’ Pages вҶ’ Source йҖүдәҶ main иҖҢйқһ gh-pagesгҖӮ main йҮҢеҸӘжңүжәҗз ҒпјҢжІЎжңүеҸҜжңҚеҠЎзҡ„ index.htmlпјҢиҮӘз„¶ 404гҖӮ

    жҺ’жҹҘдёҺдҝ®еӨҚпјҡ

    1. иҝӣд»“еә“ Settings вҶ’ PagesпјҢзңӢ Source еҪ“еүҚжҢҮеҗ‘е“ӘдёӘеҲҶж”ҜгҖӮ
    2. иӢҘдёҚжҳҜ gh-pagesпјҢж”№жҲҗ gh-pages / (root)гҖӮ
    3. зЎ®и®Ө Custom domain еЎ«зҡ„жҳҜдҪ зҡ„еҹҹеҗҚпјҲеҰӮ nav.hanvon.topпјүпјҢдё” DNS check йҖҡиҝҮ пјҲDNS йҮҢеә”жңү CNAME <дҪ зҡ„еҹҹеҗҚ> вҶ’ <user>.github.ioпјүгҖӮ
    4. д»“еә“йңҖдёә PublicпјҲз§Ғжңүд»“еә“зҡ„ Pages йңҖиҰҒ GitHub Pro/TeamпјҢе…¬ејҖйЎ№зӣ®е…Қиҙ№пјүгҖӮ
    5. ж”№е®ҢеҗҺзӯү 1вҖ“2 еҲҶй’ҹи®© Actions йҮҚи·‘гҖҒиҜҒд№ҰзӯҫеҸ‘пјҢеҶҚи®ҝй—®гҖӮ

    зңҹе®һжЎҲдҫӢпјҡжҲ‘д»¬зҡ„ works.hanvon.top дёҖејҖе§Ӣ 404пјҢе°ұжҳҜеӣ дёә Source иҜҜи®ҫдёә mainпјӣ ж”№еӣһ gh-pages еҗҺз«ӢеҚіжҒўеӨҚ HTTP 200 + HTTPSгҖӮ

    е…ӯгҖҒдёҖзңјзңӢжҮӮпјҡдёӨеҲҶж”ҜеҜ№з…§иЎЁ¶

    з»ҙеәҰ main gh-pages
    и§’иүІ жәҗз Ғ / дәәз»ҙжҠӨ жҲҗе“Ғ / жңәеҷЁз”ҹжҲҗ
    е…ёеһӢеҶ…е®№ .mdгҖҒmkdocs.ymlгҖҒconfig.tomlгҖҒlayouts/гҖҒstatic/ index.htmlгҖҒ*.cssгҖҒ*.jsгҖҒCNAME
    иғҪеҗҰзӣҙжҺҘи®ҝй—® вқҢ дёҚиғҪпјҲж— жңҚеҠЎз”ЁйҰ–йЎөпјү вң… иғҪпјҲиў« Pages serveпјү
    и°ҒжқҘеҶҷ дҪ  git push mkdocs gh-deploy / CI / жүӢеҠЁжҺЁйҖҒ
    Pages Source еә”йҖүе®ғпјҹ вқҢ еҗҰ вң… жҳҜ
    иғҪеҗҰзңӢеҲ°зҪ‘з«ҷ иҰҒзңӢжһ„е»әдә§зү© е°ұжҳҜзҪ‘з«ҷжң¬иә«

    дёғгҖҒйҮҚиҰҒдҫӢеӨ–пјҡз”ЁжҲ·йЎөпјҲUser Pagesпјүдёәд»Җд№ҲгҖҢе°ұжҳҜ mainгҖҚ¶

    дёҠйқўгҖҢжҲҗе“ҒиҗҪеңЁ gh-pagesгҖҚжҳҜйЎ№зӣ®йЎөпјҲProject Pagesпјүзҡ„й»ҳи®ӨиЎҢдёәгҖӮдҪҶ GitHub Pages иҝҳжңү дёҖз§Қзү№ж®Ҡд»“еә“вҖ”вҖ”з”ЁжҲ·йЎө / з»„з»ҮйЎөпјҲUser / Org Pagesпјүпјҡ

    • д»“еә“еҗҚеҝ…йЎ»жҒ°еҘҪжҳҜ <з”ЁжҲ·еҗҚ>.github.ioпјҲдҫӢеҰӮ wild-civil.github.ioпјүгҖӮ
    • GitHub й»ҳи®ӨзӣҙжҺҘд»Һ main/master еҲҶж”Ҝзҡ„ж №зӣ®еҪ• serveпјҢдёҚз»ҸиҝҮ gh-pagesгҖӮ
