    EDA PCB 入门 嘉立创

    嘉立创 EDA（LCEDA）¶

    嘉立创 EDA（也叫 LCEDA / EasyEDA，嘉立创收购后深度整合）是一款免费的国产 EDA 工具，最大的特点是「设计」和「制造」打通：画完原理图和 PCB，可以直接在嘉立创（JLCPCB）下单打样和 SMT 贴片，元器件直接从立创商城选型，几乎不用自己建库。

    对入门者和「画完就想拿到板子」的人来说，它是目前最顺手的选择——不用配环境、不用买授权、不用自己画封装就能出板。

    形态¶

    • 网页版：打开浏览器就能用，工程存在云端，换设备无缝衔接。
    • 客户端：Windows / macOS / Linux 独立程序，体验更顺、可离线画。

    两者工程互通。

    核心能力¶

    • 原理图（Schematic）与 PCB 一体编辑；
    • 内置立创商城元器件库，选型即带真实料号、封装、3D 模型；
    • PCB 支持多层板、铺铜、DRC、3D 预览；
    • 一键下单 PCB 与 SMT 贴片（BOM 直接喂给产线）；
    • 支持导出 Gerber / BOM / 坐标文件，也能迁移到其它工具。

    与 Altium Designer 的取舍¶

    维度 嘉立创 EDA Altium Designer
    成本 免费 订阅付费
    上手 快，开箱即用 陡，需学概念与配置
    库 商城库现成 需自建/采购库
    复杂板 够用（中低复杂） 强（高速/刚柔/BGA）
    生态 打样/SMT 一体 岗位/资料最多

    上手建议¶

    1. 从网页版起步，画一个最小系统（MCU + 电源 + 几个阻容）；
    2. 直接在库里搜料号放元件，体会「不用建库」的爽感；
    3. 画完 PCB 跑 DRC，再点「下单 PCB」走一遍真实流程；
    4. 想进阶复杂板时，再切到 Altium / KiCad。

    规划中的主题¶

    • 元件库与封装：什么时候需要自己画封装
    • 实战下单流程：板材、层数、阻焊、字符、SMT 选项怎么填
    • 规则驱动设计：嘉立创 EDA 里的 DRC 与设计规则
